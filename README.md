# Log Viewer - IT Top College Journal

WPF приложение на C# для просмотра и управления зашифрованными логами с использованием SHA-256 и AES-256 шифрования.

## Описание проекта

Log Viewer - это настольное приложение для работы с зашифрованными файлами логов. Приложение позволяет дешифровать, просматривать, редактировать и анализировать записи логов с удобным современным интерфейсом.

## Основные возможности

### 1. Работа с зашифрованными файлами
- Дешифровка файлов логов с использованием AES-256-CBC
- Поддержка SHA-256 хеширования ключей
- Автоматическое определение типа файла (логи/отчеты)

### 2. Управление записями (CRUD)
- **Добавление** - создание новых записей логов
- **Просмотр** - отображение списка с детальной информацией
- **Редактирование** - изменение существующих записей
- **Удаление** - удаление ненужных записей

### 3. Поиск и фильтрация
- Поиск по тексту (тип действия, описание, заметки)
- Фильтрация по категории (Система, Пользователь, Сеть, База данных, Безопасность, Приложение, Производительность, Другое)
- Фильтрация по статусу (Новый, В процессе, Решен, Игнорирован, Критический, Архивирован)
- Сортировка по времени (новые записи сверху)

### 4. Статусная модель
Отслеживание состояния каждой записи:
- **Новый** - новая запись, не просмотрена
- **В процессе** - запись обрабатывается
- **Решен** - проблема решена
- **Игнорирован** - запись проигнорирована
- **Критический** - высокий приоритет
- **Архивирован** - запись архивирована

### 5. Статистика и отчеты
- Общее количество записей
- Количество ошибок и критических записей
- Распределение по категориям и статусам
- Экспорт отчетов в TXT формат

### 6. Хранение данных
- Автоматическое сохранение в JSON формате
- Загрузка данных при запуске приложения
- Хранение настроек пользователя

## Запуск приложения

1. Откройте решение `IT_Web_Soft.sln` в Visual Studio
2. Восстановите NuGet пакеты
3. Соберите проект (Build)
4. Запустите приложение (F5)

## Использование

1. **Открытие файла**: Нажмите "Abrir Archivo" и выберите зашифрованный файл (.enc)
2. **Просмотр**: Выберите запись в списке для просмотра деталей
3. **Добавление**: Нажмите "Agregar" для создания новой записи
4. **Редактирование**: Выберите запись и нажмите "Editar"
5. **Удаление**: Выберите запись и нажмите "Eliminar"
6. **Фильтрация**: Используйте выпадающие списки и поле поиска
7. **Статистика**: Нажмите "Estadísticas" для просмотра отчетов
8. **Экспорт**: Нажмите "Exportar" для сохранения в TXT
